This page and the resource pack is in a state of total work in progress.

About

Minecraft Reborn is a Minecraft Mod that aims to reimagine Minecraft. It is not in development yet, and development will start once Minecraft 1.15 comes out. In the mean time, I am working on the resource pack for this mod, which adds alternative block textures. You can see images of the resource pack in the Gallery tab.



Download

If you want to download the Minecraft Reborn Resource Pack (MRRP), click here. Drag the zip file into your resource packs folder and active the resource pack.
